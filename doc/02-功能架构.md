# 功能架构

> 所有功能模块的全局视图，按阶段迭代更新。

## 功能总览

### 语音子系统

| 功能 | 描述 | 阶段 |
|------|------|------|
| 语音输入（ASR） | 语音识别转文字，替代键盘输入 | v1 |
| 语音唤醒与打断 | 唤醒词激活、说话时自动打断播放 | v1 |
| 语音回复（TTS） | AI 回复文字转语音播放 | v1 |
| 语音指令 | 自然语言指令 → 意图解析 → 调用工具/API 执行 | v2 |
| 语音操控电脑 | 通过语音操作电脑 UI（截屏识别→模拟操作） | v3 |

### 对话子系统

| 功能 | 描述 | 阶段 |
|------|------|------|
| 多轮对话 | 上下文理解的连续对话 | v1 |
| 多通道接入 | Web、移动端、CLI、飞书等 | v1 |
| 流式输出 | SSE/WebSocket 实时输出 | v1 |
| 意图理解 | NLU 解析用户意图，支持澄清追问 | v2 |

### Agent 核心能力

| 功能 | 描述 | 阶段 |
|------|------|------|
| ReAct 推理循环 | 推理→执行→观察的闭环 | v1 |
| 工具调用 | 语义匹配工具、参数填充、执行 | v1 |
| Planning 策略 | CoT / Plan-and-Solve / ReWOO 自动选择 | v2 |
| Memory 管理 | 工作记忆 / 短期记忆 / 长期记忆 | v2 |

### 知识与记忆

| 功能 | 描述 | 阶段 |
|------|------|------|
| 向量检索 | 语义相似度搜索知识库 | v2 |
| 个人知识库 | 文档导入、索引、问答 | v2 |
| 记忆持久化 | 跨会话记忆存储与检索 | v2 |
| 记忆遗忘 | 淘汰、压缩、重要性评分 | v3 |

### 多模型管理

| 功能 | 描述 | 阶段 |
|------|------|------|
| 云端模型接入 | OpenAI 兼容接口，多供应商 | v1 |
| 本地模型接入 | 本地部署模型推理 | v2 |
| 模型切换 | 同一任务可切换不同模型 | v2 |
| 模型降级 | 主模型不可用自动切换备用 | v2 |

### 运维与安全

| 功能 | 描述 | 阶段 |
|------|------|------|
| 日志与追踪 | 统一 Trace ID、全链路日志 | v1 |
| 配置管理 | YAML 配置 + 分层覆盖 | v1 |
| 权限控制 | 工具调用权限、用户认证 | v2 |
| Prompt 防护 | 防注入、防越权 | v3 |
